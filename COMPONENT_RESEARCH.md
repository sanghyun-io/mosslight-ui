# Component Research

This pass compared common component surfaces from established libraries and selected primitives that fit Mosslight UI's warm, practical design language.

## Sources

- Chakra UI components overview: broad component catalog including Accordion, Alert, Avatar, Breadcrumb, Checkbox, Field, Pagination, Progress, Radio, Separator, Skeleton, Slider, Spinner, Table, Tabs, Toast, Tooltip.
- shadcn/ui components: open-code component catalog including Accordion, Alert, Avatar, Badge, Breadcrumb, Button, Card, Checkbox, Dialog, Field, Pagination, Progress, Radio Group, Select, Separator, Skeleton, Slider, Spinner, Switch, Tabs, Toast, Tooltip.
- Ant Design components overview: mature application component set including Button, Divider, Breadcrumb, Pagination, Checkbox, Form/Input, Radio, Switch, Slider, Select, Progress, Table, Tooltip, Tabs, Alert, Avatar, Skeleton.
- Radix Primitives overview: accessibility-focused primitives for common patterns such as accordion, checkbox, dialog, select, slider, and tooltip.

## Selection

Mosslight UI now covers the practical starter surface expected from a small public React component library:

- Actions: `Button`
- Status and feedback: `Alert`, `Badge`, `Progress`, `Spinner`, `Toast`
- Containers and disclosure: `Accordion`, `Card`, `Dialog`, `Tabs`, `Tooltip`
- Forms: `Checkbox`, `Field`, `FormControl`, `RadioGroup`, `Select`, `Slider`, `Switch`
- Navigation: `Breadcrumb`, `Pagination`
- Data and layout helpers: `Avatar`, `Separator`, `Skeleton`
