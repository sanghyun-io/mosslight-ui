# Public Repository Checklist

Mosslight UI can be public as an original cozy fantasy design system.

Keep these boundaries before publishing:

- Do not use anime, manga, game, studio, character, logo, screenshot, or soundtrack assets unless you have explicit rights.
- Do not market the package with a trademarked title in the repo name, package name, preview images, tags, or README headline.
- Keep the description mood-based: cozy fantasy, moss, parchment, sky, quiet motion, inked borders.
- The repo uses the MIT license. Change `LICENSE` before publishing if a different reuse model is needed.
- Review generated screenshots before committing; they should show only original UI assets.

Suggested public repo description:

> Warm fantasy React UI primitives with moss, parchment, sky accents, tactile borders, and restrained ambient motion.
